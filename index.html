Wire-Pod Installation UI

It is recommended you read README.md

if you have configured everything here, just click the start button to start the installation

If you are unsure what to do, just select the recommended option.

Basic installation settings


Language model settings

Here you can choose between two voice recognition models.
The first one has a large vocabulary, it is faster, less accurate and has about 100 MB
The second one has a huge vocabulary, it is slower, more accurate but handles fast speech better, it has about 900 MB


Wire-Pod Domain Name or IP

Would you like to use an IP address, domain or a hostname for the Wire-Pod? Or would you like to use escapepod.local?
The IP address is recommended for OSKR Vectors
escapepod.local is recommended for production Vectors.


Webserver HTTP Port

What port would you like to set for the HTTP webserver used for custom intents, configuration, etc.


Weather API

Would you like to setup weather commands?This involves creating a free account at weatherapi.com and putting in your API key here. This is optional.



Knowledge Graph

Would you like to setup Knowledge Graph ("I have a question")? This involves creating a free account at houndify.com and putting in your Client Key and Client ID here.
Note: It may look like you only get a trial, but there is an actual free tier with 100 free requests per day.
This is optional.




Temperature Unit

What temperature unit would you like to use?
