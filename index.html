Basic installation settings


Language model settings

Here you can choose between two voice recognition models.
The first one has a large vocabulary, it is faster, less accurate and has about 100 MB
The second one has a huge vocabulary, it is slower, more accurate but handles fast speech better, it has about 900 MB


Wire-Pod Domain Name or IP

Would you like to use an IP address, domain or a hostname for the Wire-Pod? Or would you like to use escapepod.local?
The IP address is recommended for OSKR Vectors
escapepod.local is recommended for production Vectors.
